{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} ContainerSSH is an SSH server that launches Docker containers or Kubernetes pods for every new connection and sends the users directly into this container. Using an easy to customize API you can integrate your own authentication system and make decisions about which container to launch with what parameters on the fly. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fas fa-globe" title="For Web Hosting" url="https://github.com/janoszen/containerssh" %}}

Provide full SSH access to your users while limiting access to only the sites they are allowed to.
